## 🔴 Critical — 색상/토큰 위반

### 1. `src/app/page.tsx` (Next.js 기본 생성 파일)
Next.js 프로젝트 초기화 시 자동 생성된 파일로, 디자인 시스템 토큰이 전혀 적용되지 않았습니다.

| Line | 위반 내용 | 원본 | 권장 대체 |
|------|-----------|------|-----------|
| 39 | `hover:bg-[#383838]` | `#383838` = `neutral-900` | `hover:bg-neutral-900` |
| 39 | `dark:hover:bg-[#ccc]` | `#ccc` ≈ `neutral-400` | `hover:bg-neutral-400` |
| 54 | `dark:hover:bg-[#1a1a1a]` | `#1a1a1a` ≈ `neutral-1100`에 근접 | `hover:bg-neutral-1000` |
| 54 | `border-black/[.08]` | opacity 수식 사용 | 토큰 미정의 (무시 가능, Next.js 기본) |

> **→ 권장 조치**: `page.tsx`는 Next.js 기본 파일이므로 우선순위 낮음. 향후 랜딩 페이지로 교체 시 전면 재작성 필요.

### 2. `src/app/design-system/page.tsx` — `CodeBlock` 내부 예시 문자열
| Line | 내용 |
|------|------|
| 392–394 | `bg-[#2c2c2c]`, `text-[#1e1e1e]`, `border-[#d9d9d9]` |

> ⚠️ 이것은 **"금지된 패턴 안내용 예시 코드 문자열"** 이므로 실제 클래스 적용이 아닙니다.  
> **→ 위반 아님 (false positive)** — `<code>` 블록 안에 렌더링되는 텍스트임.
